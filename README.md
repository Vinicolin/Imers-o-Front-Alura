# 🎧 Imersão Front-End - Alura

Projeto desenvolvido durante a **Imersão Front-End da Alura**, com foco na construção de interfaces modernas utilizando HTML, CSS e JavaScript.

---

## 📌 Sobre o projeto

Este projeto tem como objetivo colocar em prática conceitos fundamentais de desenvolvimento front-end, como:

- Estruturação de páginas com HTML
- Estilização com CSS
- Responsividade
- Manipulação de elementos com JavaScript

A proposta é recriar uma interface inspirada em plataformas modernas (como streaming/música), aplicando boas práticas de desenvolvimento web.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## ▶️ Como executar o projeto

1. Clone o repositório:
```bash
git clone https://github.com/Vinicolin/Imers-o-Front-Alura.git
```

2. Acesse a pasta:
```bash
cd Imers-o-Front-Alura
```

3. Abra o arquivo `index.html` no navegador
